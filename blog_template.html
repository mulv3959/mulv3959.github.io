Favorite Websites

ESPN

To be honest the main reason why I like ESPN is probably because I am used to it. I have been using it for years and the layout has stayed relatively the same. Right when you enter the site your eyes are immediatly drawn to the slide show that shows everything that is happening today in the sports world. I would not change anythign with the layout again mainly because I am used to it but the content can be iffy sometimes.

USA Today

I only recently started looking at the web site but I love how it is layed out. They have pretty much ever topic possible (Sports, Tech, US News, World News, Weather, etc) and it is incredibly easy to navigate through them all.

Grantland Sports

Grantland primarily is in here for the content. They take a little different stance on covering sports news which I really enjoy. They often use advanced stats, add a little humor, and cover the smaller markets much more often than a power house like espn. The layout could be a little better as they often use nicknames in titles making it hard for new visitors to navigate the site.

ID’s v Classes

ID’s and Classes are used so you can identify a section of a website with a name of your choosing. Later, when you would like to add color or a design to this section, you “call” on it by using the name you chose above. When designing a website this can be very useful. So what is the difference between an id and a class?

ID

An id can only be used to identify ONE section. I like to think of my own personal ID (or drivers license) when trying to remember what an id is in html/css. My driver’s license only identifies me, I cannot use it to identify anyone else because it is specific to me. Similar to how an id works in html/css; it can only be used identify one specific section.

Class

A class on the other hand can be used to identify multiple sections that share a similar property. I usually think of a school classroom, lets say a Math Class. Each person within this math class is a different section of the website. So the every student in the classroom can fall under the same class, “math class.”
